# 플래시카드 질문 컨벤션

- question 필드는 구어체 한국어로 작성
- 끝맺음: ~나요?, ~가요?, ~뭔가요? 등 자연스러운 의문형
- ✓ "타입 추론이란 뭔가요?"
- ✓ "호이스팅이 뭔가요?"
- ✗ "타입 추론이란?"
- ✗ "타입 호환성(Type Compatibility)과 구조적 타이핑이란?"

# 플래시카드 답변 안티패턴

답변이 AI가 쓴 것처럼 느껴지게 만드는 패턴 목록. 아래 표현은 피하세요.

- `~을 통해 달성할 수 있습니다` → `~하면 됩니다`
- `첫째, 둘째, 셋째` 나열 구조 → 자연스러운 산문으로 연결
- `결론적으로`, `요약하자면` → 생략하거나 바로 결론을 말할 것
- `~라고 볼 수 있습니다` → `~입니다`로 단정
- `~할 수 있습니다` 반복 남발 → 한 번만 쓰거나 단정형으로
- 모든 문장이 비슷한 길이 → 짧은 문장과 긴 문장을 섞어 리듬 변화를 줄 것
